## 🚀 快速開始

### 環境需求

```bash
Python 3.8+
攝像頭設備
Windows/macOS/Linux
```

### 安裝與執行

```bash
# 1. 安裝依賴
pip install -r requirements.txt

# 2. 啟動程式
python app.py

# 3. 使用方式
# - 點擊「啟動」開始追蹤
# - 食指在綠色框內移動控制滑鼠
# - 按空白鍵進行左鍵點擊
```

## 📈 技術架構展示

### 系統組件

```
🧠 MediaPipe (深度學習)
├── BlazePalm 手部檢測
└── 21點關鍵點追蹤

👁️ OpenCV (電腦視覺)  
├── 影像擷取與處理
├── 座標轉換
└── 視覺化呈現

🖱️ 滑鼠控制 (系統API)
├── pyautogui 游標控制
├── keyboard 全域監聽
└── 平滑演算法

🎨 使用者介面 (GUI)
├── tkinter 圖形介面
├── 多線程處理
└── 即時參數調整
```

### 處理流程

```
攝像頭 → 影像預處理 → 手部檢測 → 關鍵點提取 → 手勢識別 → 座標映射 → 滑鼠控制
   ↓         ↓          ↓        ↓         ↓        ↓        ↓
 硬體層    視覺層     AI層     特徵層    邏輯層    轉換層    控制層
```
